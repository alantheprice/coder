# Simple Server Project

A basic HTTP server implementation in Go.

## Features

- Basic HTTP routing
- JSON status endpoint
- User management (planned)

## Running

```bash
go run *.go
```

## Endpoints

- `/` - Home page
- `/status` - Server status
- `/users` - User management (not implemented)
